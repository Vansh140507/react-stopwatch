## ✨ Features

- 🎯 **Precise Time Tracking** — Displays time in `HH:MM:SS` format with zero-padding for clean readability.
- ⏯️ **Start / Stop Controls** — Toggle timing with a single click, preventing duplicate intervals.
- 🔄 **Reset Functionality** — Instantly reset the timer back to `00:00:00` with proper cleanup.
- 🧹 **Memory-Safe** — Uses `useRef` for interval management, avoiding memory leaks and stale closures.
- 💅 **Modern UI** — Clean, minimal design with rounded corners, flex layout, and hover interactions.
- 📱 **Responsive** — Adapts seamlessly across desktop and mobile viewports.
